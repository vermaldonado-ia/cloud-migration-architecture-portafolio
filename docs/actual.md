# 🏗️ Arquitectura Actual (AS-IS)

* Máquina virtual por cliente
* Base de datos independiente por cliente
* Plataforma web personalizada

Problemas:

* Múltiples versiones del sistema
* Cambios en producción sin control
* Sin ambientes de desarrollo ni QA
* Riesgo alto de pérdida de datos
* Baja escalabilidad
