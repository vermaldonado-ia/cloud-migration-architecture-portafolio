# 🧩 Contexto del Proyecto

Plataforma ERP logística orientada a la gestión de importaciones y exportaciones.

Modelo de negocio:

* SaaS (arriendo de plataforma)
* Personalización por cliente

Industria:

* Comercio exterior

Problema principal:

* Pérdida de información
* Altos costos operativos
* Infraestructura fuera del core del negocio

Objetivo:

* Migrar a cloud
* Reducir costos
* Mejorar estabilidad
* Escalar internacionalmente
